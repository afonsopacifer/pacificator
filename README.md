# Pacificator.js

> Pacificate your CSS

## What does it do?

**Before**
```css
.class{
color:#fff;}
```

**After**
```css
.class {
  color: #fff;
}
```

## How to install

```sh
$ npm install -g pacificator
```

## How to use

```sh
$ pacificator style.css
```
